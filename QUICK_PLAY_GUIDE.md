# ⚡ 3步开始游戏！

## 第1步：回到Unity编辑器 🖥️

确保你在Unity Editor中，而不是在播放模式。

## 第2步：打开菜单 📋

```
顶部菜单栏 → "游戏Demo" → "升级为可玩Demo"
```

会弹出一个窗口。

## 第3步：点击升级按钮 🚀

```
点击大按钮："🚀 升级为可玩Demo"
```

等待几秒钟，会提示完成。

---

## 然后：点击播放 ▶️

```
Unity顶部工具栏 → 播放按钮 ▶️
```

## 游戏控制 🎮

```
W A S D    = 移动
Shift      = 加速
ESC        = 退出
```

---

## 你会看到：

- ✅ 白色胶囊（玩家）- 可以移动
- ✅ 红色方块（敌人）- 自动生成，每2秒1个
- ✅ 左上角 - 控制说明
- ✅ 右上角 - 游戏状态

---

## 如果菜单找不到？

**等待1-2分钟让Unity编译完成**，然后再试。

或者查看详细教程：`PLAYABLE_DEMO_GUIDE.md`
